# Github Configuration

This repository provides Github configuration and defaults for the
[explainers-by-googlers](https://github.com/explainers-by-googlers) org.

Among other things, it holds the [default community health
files](https://docs.github.com/en/communities/setting-up-your-project-for-healthy-contributions/creating-a-default-community-health-file#supported-file-types).
